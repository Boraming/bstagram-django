# bstagram-django



# 소개	
인스타그램과 같이  회원가입을 한 사람은 사진을 올리고 사이트의 게시글을 읽을 수 있으며 댓글을 달 수 있는 커뮤니티 사이트입니다.



# 앱설명
## account
 - 회원가입
 - 로그인
 - 로그아웃


## bstagram
- Nav, Footer을 포함한 공통 UI (base.html)


## photo(main)
- 게시물(사진, 글), 댓글작성가능
- 상세페이지
- 업로드
- 업데이트



# 개발환경

 - Windows 10
 - Visual Studio Code
 - Python (Django)
